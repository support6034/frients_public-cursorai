- Frontend 빌드: 진행 중
- Backend 테스트: 필요

### 배포 준비
- Git 저장소 확인: 필요
- 배포 플랫폼 계정: 필요

1. **Vercel 계정** (Frontend 배포용)

2. **Railway 계정** (Backend 배포용)
   - 이메일 또는 GitHub 연동

3. **Git 저장소 정보**
   - 저장소 URL 확인
   - 브랜치 정보 확인

4. **환경 변수**
   - 백엔드 URL (Railway 배포 후)
   - CORS 설정 (프론트엔드 URL)
