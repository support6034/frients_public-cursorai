### 1.3 템플릿 선택

| 테스트 항목 | 절차 | 예상 결과 | 실제 결과 | 비고 |

| 기본 템플릿 자동 선택 | 페이지 로드 | ID 1,2,3,4,5 템플릿 자동 선택 (체크 표시) | ✅ 통과 | |
| 템플릿 카드 클릭 | 템플릿 카드 클릭 | 체크 표시 + 배경색 변경 (#f0f4ff) | ✅ 통과 | |
| 템플릿 해제 | 선택된 템플릿 재클릭 | 체크 해제 + 배경색 원복 | ✅ 통과 | |
| 템플릿 이름 표시 | 각 템플릿 확인 | 주문접수, 결제완료 등 이름 표시 | ✅ 통과 | |
| 템플릿 설명 표시 | 각 템플릿 확인 | "주문이 접수되었습니다" 등 설명 표시 | ✅ 통과 | |
| 기본 배지 표시 | 기본 템플릿 확인 | "기본" 배지 표시 (녹색) | ✅ 통과 | |

- ✅ 상품준비중 (기본, 선택됨)
- ✅ 배송시작 (기본, 선택됨)
- ✅ 배송완료 (기본, 선택됨)
- ✅ 구매확정 (선택 안됨)
- ✅ 리뷰요청 (선택 안됨)
- ✅ 재고부족 (선택 안됨)
- ✅ 주문취소 (선택 안됨)
- ✅ 환불완료 (선택 안됨)
