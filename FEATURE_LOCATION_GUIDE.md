## 📍 구현된 기능 위치

### 1. 대시보드 탭 (`/ai-bot` - 기본 탭)

**위치**: 페이지 상단 "대시보드" 탭 클릭 또는 기본 표시

**구현된 기능**:

#### 1.1 업종 선택 카드 (4개)
- **위치**: 대시보드 상단
- **기능**: 
  - 🛒 쇼핑몰 (활성화됨)
  - 📅 예약 (준비중)
  - 🏢 프랜차이즈 (준비중)
  - 🏪 매장 (준비중)
- **파일**: `frontend/src/components/ai-alimbot/AIAlimbotDashboard.vue` (라인 8-19)

#### 1.2 템플릿 그리드 (10개)
- **위치**: 업종 선택 카드 아래
- **기능**: 
  - 10개 템플릿 카드 표시
  - 템플릿 클릭으로 선택/해제
  - 선택된 템플릿은 보라색 배경으로 표시
  - 기본 템플릿 5개 자동 선택 (주문접수, 결제완료, 상품준비중, 배송시작, 배송완료)
- **템플릿 목록**:
  1. 주문접수 (기본)
  2. 결제완료 (기본)
  3. 상품준비중 (기본)
  4. 배송시작 (기본)
  5. 배송완료 (기본)
  6. 구매확정
  7. 리뷰요청
  8. 재고부족
  9. 주문취소
  10. 환불완료
- **파일**: `frontend/src/components/ai-alimbot/AIAlimbotDashboard.vue` (라인 21-44)

#### 1.3 템플릿 저장 버튼

- **기능**: 선택한 템플릿 저장 및 워크플로우 동기화
- **파일**: `frontend/src/components/ai-alimbot/AIAlimbotDashboard.vue` (라인 39-43)

## ⚠️ 현재 문제점

브라우저에서 페이지가 제대로 표시되지 않는 경우:

### 가능한 원인
1. **백엔드 서버 미실행**
   - Backend 서버가 포트 5000에서 실행되지 않음
   - 해결: `cd backend && npm start`

2. **프론트엔드 서버 미실행**
   - Frontend 서버가 포트 5173에서 실행되지 않음
   - 해결: `cd frontend && npm run dev`

3. **컴포넌트 로드 오류**
   - 브라우저 콘솔에서 에러 확인 필요
   - 컴포넌트 import 경로 문제 가능

4. **스타일 미적용**
   - CSS 파일이 로드되지 않음
   - Tailwind CSS 설정 문제 가능
