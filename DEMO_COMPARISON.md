## 3. 데모 시나리오

### 시나리오 1: 대시보드 탭 확인

**원본 (https://workflow.changups.kr)**:
1. "AI 알림봇" 메뉴 클릭
2. 대시보드 탭 기본 표시
3. 업종 선택 카드 4개 표시 (쇼핑몰 활성화)
4. 템플릿 그리드 10개 표시
5. 기본 템플릿 5개 자동 선택 (주문접수, 결제완료, 상품준비중, 배송시작, 배송완료)
6. 디버그 로그 뷰어 하단 표시

**신규 (http://localhost:5173/ai-bot)**:
- [ ] 동일한 UI 표시 확인
- [ ] 템플릿 선택/해제 동작 확인
- [ ] "선택한 템플릿 저장" 버튼 동작 확인

### 시나리오 2: 설정 탭 확인

**원본**:
1. "설정" 탭 클릭
2. 결제 섹션 표시 (잔액, 충전 금액 선택, 결제 수단)
3. 연동 섹션 표시 (스마트스토어 연동 활성화 체크박스)
4. 템플릿 선택 섹션 표시 (10개 템플릿 체크박스)

**신규**:
- [ ] 동일한 UI 표시 확인
- [ ] 연동 설정 저장 동작 확인
- [ ] 템플릿 선택 저장 동작 확인

### 시나리오 3: 워크플로우 동기화 확인

**원본**:
1. 템플릿 저장 후 "마케팅자동화" 메뉴 이동
2. `[AI알림봇] 주문접수` 등 워크플로우 생성 확인
3. 워크플로우 편집에서 이벤트명, 웹훅 URL 확인

**신규**:
- [ ] 워크플로우 동기화 동작 확인
- [ ] 생성된 워크플로우 확인
