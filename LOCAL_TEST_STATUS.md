**원인 추정**:
- 실제 실행 중인 앱이 다른 설정을 사용 중일 수 있음
- 또는 라우터 설정이 제대로 로드되지 않음
- 또는 다른 라우터 설정 파일이 사용 중일 수 있음

## 시험 진행 불가 상태

현재 상태에서는 신규 버전의 기능 시험을 진행할 수 없습니다.

**이유**:
1. `/ai-bot` 경로가 라우터에서 매칭되지 않음
2. AI 알림봇 페이지에 접근할 수 없음
3. 실제 렌더링되는 앱이 예상과 다름

1. **라우터 문제 해결**

2. **실제 실행 중인 앱 확인**
