# cursorAI 배포 진행

## 배포 진행 중

### 필요한 정보

**Railway 배포용:**
- Railway 계정 접근 권한 또는 Railway Token
- 또는 Railway 대시보드에서 프로젝트 생성 완료 여부

**Vercel 배포용:**
- Vercel 계정 접근 권한 또는 Vercel Token
- 또는 Vercel 대시보드에서 프로젝트 생성 완료 여부

### 배포 방법

**방법 1: CLI 사용 (자동)**
- Railway CLI: `railway up`
- Vercel CLI: `vercel --prod`

**방법 2: 대시보드 사용 (수동)**
- Railway: 웹 대시보드에서 배포
- Vercel: 웹 대시보드에서 배포
