### 2. 설정 탭 기능

**접근 방법**: 상단 "설정" 탭 클릭

#### 2.1 결제 설정 섹션
- **위치**: 설정 탭 첫 번째 섹션
- **제목**: "1. 결제"
- **표시 내용**:
  - 잔액 표시
  - 충전 금액 선택 (10,000원 / 50,000원 / 100,000원)
  - 결제 수단 선택
  - 충전 내역 테이블
- **파일**: `frontend/src/components/PaymentSettings.vue`

#### 2.2 연동 설정 섹션
- **위치**: 설정 탭 두 번째 섹션
- **제목**: "2. 연동"
- **표시 내용**:
  - 스마트스토어 연동 활성화 체크박스
  - API Key 입력 필드
  - API Secret 입력 필드 (비밀번호 타입)
  - 스토어 ID 입력 필드
  - "연동 설정 저장" 버튼
- **파일**: `frontend/src/components/ai-alimbot/AIAlimbotSettings.vue` (라인 12-61)

#### 2.3 템플릿 선택 섹션
- **위치**: 설정 탭 세 번째 섹션
- **제목**: "3. 템플릿 선택"
- **표시 내용**:
  - 10개 템플릿 체크박스 목록 (세로 나열)
  - 각 템플릿에 이름과 설명 표시
  - 선택된 템플릿은 보라색 배경
  - "템플릿 선택 저장" 버튼
- **파일**: `frontend/src/components/ai-alimbot/AIAlimbotSettings.vue` (라인 63-90)

## 🛠️ 문제 해결

### 현재 발생 중인 문제
- Vue Router가 `/ai-bot` 경로를 찾지 못함
- 컴포넌트가 로드되지 않음

### 해결 방법

1. **프론트엔드 서버 재시작**
```bash
cd C:\Users\hckim\frients_public\cursorAI\frontend
npm run dev
```

2. **브라우저 강제 새로고침**
- Ctrl + F5
- 브라우저 캐시 클리어

3. **컴포넌트 파일 확인**
- `frontend/src/views/AIAlimbotView.vue` 파일 존재 확인
- 파일 내용 확인
