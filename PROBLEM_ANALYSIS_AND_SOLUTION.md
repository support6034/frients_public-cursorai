## 발견된 문제

### 1. 라우터 경로 불일치
- **브라우저에서 보이는 네비게이션**: `/`, `/marketing`, `/lists`
- **`App.vue`에 정의된 링크**: `/`, `/marketing-automation`, `/ai-alimbot`
- **라우터 설정**: `/marketing-automation`, `/ai-alimbot` 등

### 2. Vue Router 경고
- `[Vue Router warn]: No match found for location with path "/ai-alimbot"`
- `/ai-alimbot` 경로가 매칭되지 않음

### 3. 실제 실행 중인 앱 불일치
- 브라우저에서 보이는 네비게이션이 `App.vue`의 것과 다름
- Vue 인스턴스는 있지만 Vue Router가 제대로 작동하지 않음
